# Routing

## Routing List

- `/` - トップページ
- `/about` - Aboutページ
- `/achievements` - 実績一覧ページ
- `/achievements/:id` - 実績詳細ページ
- `/contact` - Contactページ
- `/projects` - プロジェクト一覧ページ
- `/projects/:id` - プロジェクト詳細ページ
- `/blog` - ブログ一覧ページ
- `/blog/:id` - ブログ詳細ページ
- `/skills` - スキル一覧ページ
- `/skills/:id` - スキル詳細ページ

## Routing Details

- `/` - トップページ
  - Aboutページよりも抽象的な情報を提供します。
- `/about` - Aboutページ
  - 自己紹介や経歴などの詳細情報を提供します。
- `/achievements` - 実績一覧ページ
  - これまでの実績を一覧表示します。
- `/achievements/:id` - 実績詳細ページ
  - 特定の実績に関する詳細情報を表示します。
- `/contact` - Contactページ
  - お問い合わせフォームや連絡先情報を提供します。
- `/projects` - プロジェクト一覧ページ
  - これまでに携わったプロジェクトを一覧表示します。
- `/projects/:id` - プロジェクト詳細ページ
  - 特定のプロジェクトに関する詳細情報を表示します。
- `/blog` - ブログ一覧ページ
  - ブログ記事の一覧を表示します。
- `/blog/:id` - ブログ詳細ページ
  - 特定のブログ記事の内容を表示します。
- `/skills` - スキル一覧ページ
  - 保有するスキルを一覧表示します。
- `/skills/:id` - スキル詳細ページ
  - 特定のスキルに関する詳細情報を表示します。
